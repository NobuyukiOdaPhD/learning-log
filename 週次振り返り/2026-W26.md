### ミッションを通じて
- (復習)新しいレポジトリの作成
1. GitHubでレポジトリを作成する
2. ローカルで対応するディレクトリを作成する
3. git initでレポジトリを作成する
4. あとはgithubのcommand lineでの接続にしたがう(下記は例)
```
echo "# js-playground" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:NobuyukiOdaPhD/js-playground.git
git push -u origin main
```
