### ミッションを通じて
#### (復習)新しいレポジトリの作成
1. GitHubでレポジトリを作成する
2. ローカルで対応するディレクトリを作成する
3. git initでレポジトリを作成する
4. あとはgithubのcommand lineでの接続にしたがう(下記は例)
```
echo "# js-playground" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:NobuyukiOdaPhD/js-playground.git
git push -u origin main
```
#### 初めてのJS: ボタンを押して文字を変更
- JSのコードをHTMLファイルの中に記述する際には、<script></script>の中に記述する
- const text = document.querySelector('#title'): テキストにh1要素を代入
 - "= document.getElementById"も使えるが、"document.querySelector"にすると、CSSのセレクタと同じ書き方で取得できる。(IDで取得する場合は'#id名'、クラス名で取得する場合は'.クラス名')
- button.addEventListener('click', () => {
        text.textContent = 'JS-Playground Updated!!';
      });
 - ボタンを押すことによって、テキストが記述されたものに変更される。
- JSはコマンドの終わりに';'をつける
- 「掴む → 何かしたら → 書き換える」の記述は、"querySelector -> addEventListener -> textContent"ということ。
