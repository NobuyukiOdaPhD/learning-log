#### ドットインストール - CSS, footer, section
- CSS基本構文
 - セレクタ {
  プロパティ: プロパティの値;
 }
 - 例
   img {
    border-radius: 20%;
   }
- CSSを記述する方法はいくつかある
1. <head></head>内部に<style></style>を入れる

- 使用したプロパティ
* "border-width: 1px;": 枠線の幅
* "border-width: gray;": 枠線の色
* "border-style: solid;": 枠線の種類(この場合、実戦)
* "border-radius: 8px;": 枠線の角の丸め
* "padding: 16px;": 枠線の内側に余白を設定する(webサイトでは8の倍数を使うと揃いやすいと言われる)
* "margin-bottom: 8px;" 枠線の外側に余白を設定する(この場合は、下のみ設定)
* "text-align: center;": 中央揃いにする
* "background-image: url(「ファイルのパス」);": 背景画像を設定する
- <footer></footer>: フッター表記を行う
- <section></section>: セクションを区切るための汎用タグ(使用例ではフッターをメインセクションから外すために利用)
