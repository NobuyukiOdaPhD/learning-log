### 個別に学んだこと - HTML
#### ミッション
- CSSファイルの読み込み
* <head></head>内部にCSSファイルのリンクを入れる -> <link rel="stylesheet" href="style.css">
- 使用したCSSのプロパティ
* "font-family: 'メイリオ', 'Arial', sans-serif;": フォントの指定。Macはメイリオが標準ではないため、'Hiragino Sans'で指定した方が良い
* "max-width: 700px; margin: 0 auto;": 横方向に中央寄せして読みやすい幅にする。-- ただし、画像ファイルは独立して、その幅で表示されているので、700pxをはみ出している状態。-> 画像にmax-width: 100%;を足すと枠の中に収まる。
* "color: #2738F5;": 色指定。ここではHEX表示している。色見本はこちら(https://htmlcolorcodes.com/)
* "line-height: 1.6;": 行間を決める(ここでは1.6倍)

#### ドットインストール - CSS, footer, section
- CSS基本構文
 - セレクタ {
  プロパティ: プロパティの値;
 }
 - 例
   img {
    border-radius: 20%;
   }
- CSSを記述する方法はいくつかある
1. <head></head>内部に<style></style>を入れる
- 使用したプロパティ
* "border-width: 1px;": 枠線の幅
* "border-width: gray;": 枠線の色
* "border-style: solid;": 枠線の種類(この場合、実戦)
* "border-radius: 8px;": 枠線の角の丸め
* "padding: 16px;": 枠線の内側に余白を設定する(webサイトでは8の倍数を使うと揃いやすいと言われる)
* "margin-bottom: 8px;" 枠線の外側に余白を設定する(この場合は、下のみ設定)
* "text-align: center;": 中央揃いにする
* "background-image: url(「ファイルのパス」);": 背景画像を設定する
- <footer></footer>: フッター表記を行う
- <section></section>: セクションを区切るための汎用タグ(使用例ではフッターをメインセクションから外すために利用)
